# React App
React - Node - Javascript ES6

App that manages auto-evaluations of daily objectives
It allows the user the registration of present or past objectives.
With each registration it updates the statistics automatically in real time

Login data:

user: `jaime`

passw: `jaime`


### To start application
Run this command in the directory with the package.json file 
`npm start`
